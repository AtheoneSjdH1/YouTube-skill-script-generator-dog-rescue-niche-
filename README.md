# YT Dog Rescue Shorts — AI Skill

An AI skill that turns a simple request like *"make me a video"* into a complete, ready-to-produce **YouTube Shorts package** for a dog-rescue channel — no blank-page guesswork.

Give it a prompt, and it outputs the four things you actually need to publish an 8-second Short:

- 🎬 **A Veo 3.1 Fast prompt** — a six-block, image-to-video prompt (Style / Scene / Subjects / Action / Camera / Audio) written for a single 8-second native clip with synced audio.
- 📝 **A title** — built from proven Shorts hook formulas (action-driven, not slow-burn).
- 💬 **A caption** — with hashtags, matched to the channel's voice.
- 📌 **A pinned creator comment** — for the rescue-awareness community.

## What it's for

The channel makes AI-generated dog-rescue Shorts (rendered in Veo 3.1 Fast via Google Flow), aimed at fast, strike-free monetization. This skill bakes in the hard-won rules for that: strong hook in the first two seconds, a character in motion from frame one, mute-friendly captions, loop-ready endings, and platform-compliant content (no gore, warmth-forward resolutions).

## How it works

Point your AI assistant at `SKILL.md` and ask it to make a video. The skill:

1. Reads its rule and reference files (see note below).
2. Picks a fresh rescue concept and writes the four-part package.
3. Saves each video as a numbered file so the channel builds a consistent library.

## Note on this public copy

This repository publishes **`SKILL.md` only** — the skill's instructions and format.

The private reference files it points to (real example scripts, the channel's analytics snapshots, the idea bank, and the per-video logs) are **not included**, since they contain channel-specific data. To run the full skill end-to-end you'd supply your own `references/` files following the structure described inside `SKILL.md`.
